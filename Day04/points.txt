lambdas
-------
are anonymous functions with a single line code

exp_result = lambda var1, var2, var3....: expression using these var


Comprehensions
--------------
iterating through a collection(list and dictionary) using lambda syntaxes



Scopes in Python
----------------
1. global		- var declared outside the function 
2. local 		- var declared inside the function is local (lexical scope)
3. non local

Decorator
---------
changing the behaviour of a function without touching the code inside it.


def sub(a, b):
	return a - b



sub should always return a positive result

sub(10, 5)  = 5
sub(5, 10)  = 5
